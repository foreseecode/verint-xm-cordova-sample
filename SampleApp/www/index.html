This example shows how to use checkEligibility, showInvite, showDigitalSurvey and resetState methods.

Use checkEligibility to check if the user is eligible for a survey. If the user meets trigger criteria and is in the sampling pool, the invitation will be presented.

Use resetState to reset the state of the tracker and (if applicable) reactivate the Predictive module.

Use showInvite to present (programmatically) the invitation for a given survey ID (sid).

Use showDigitalSurvey to present (programmatically) the digital survey for a given digital survey ID (sid).
